# Getting Started

## Requirements

- WordPress `7.0+`
- PHP `8.1+`
- Node.js and npm for asset builds

## Install

```bash
npm install
npm run build
```

Then copy the plugin directory into `wp-content/plugins/` and activate Dewey in wp-admin.

## Development

```bash
npm run start
```

This runs the webpack watcher via `@wordpress/scripts`.
